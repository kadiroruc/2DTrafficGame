# 2DTrafficGame

2 Boyutlu Trafik Oyunu.

Start butonuna bastığınızda oyun başlar.
Stop butonuna bastığınızda oyun biter.

Oyun içerisinde herhangi bir araca çarpdığınızda ve yolun solundan ve sağından dışarı çıktığınızda oyun sonlanır.
